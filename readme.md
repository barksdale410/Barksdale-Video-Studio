# 🎬 BARKSDALE VIDEO STUDIO

**Script to Screen — AI-Powered Movie Production**

## Quick Start
1. Paste your script
2. Select director, genre, mood
3. Click "Analyze Script" → see storyboard
4. Click "Generate Movie" → get your video

## Deployment
- Backend: Render (Python FastAPI)
- Frontend: GitHub Pages

## Tech Stack
- Frontend: HTML/CSS/JS
- Backend: Python FastAPI
- Video: NVIDIA Cosmos
- Voice: ElevenLabs
